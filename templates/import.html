=== RECIPE ===

=== STEPS ===

2. ...

=== INDEX_TERMS ===
Term1; Term2; ...
=== END ===

  • Use UTF-8 (avoid weird characters).
  • Keep section headers exactly: === INGREDIENTS ===, === STEPS ===.
  • Numbered lines help parsing: 1., 2. …
  • If preview looks wrong → fix text → preview again.
You can use either file upload or paste text (not both).
